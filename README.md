# Junseok account-book 
Django로 만든 개인 가계부 웹 서비스입니다.
수입/지출을 기록하고 카테고리별 통계를 파이차트로 시각화합니다.

---

# 주요 기능

기록 기능
- 수입 지출 기록
- 카테고리 선택
- 금액, 날짜 선택
- 최신순 정렬

수정 및 삭제 기능
- 기록 수정
- 삭제 전 confirm 팝업

통계 기능
- 카테고리별 지출/수입 합계 계산
- 표 형태로 정리
- 파이 그래프로 시각화
